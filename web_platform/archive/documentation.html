Authentication

CardioPredict API uses API keys for authentication. Include your API key in the request headers.

HTTP
Authorization: Bearer YOUR_API_KEY
Content-Type: application/json
Getting an API Key: Contact our team at api@cardiopredict.org for research or clinical API access.

Quick Start

Get started with CardioPredict in minutes using our simple REST API.

5-Minute Integration
Bash
curl -X POST "https://api.cardiopredict.org/v1/predict" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "biomarkers": {
      "crp": 2.1,
      "pf4": 12.5,
      "troponin_i": 0.012,
      "bnp": 45
    },
    "environment": "space",
    "mission_duration": 14
  }'
Python
import requests

# Configure API
api_key = "YOUR_API_KEY"
base_url = "https://api.cardiopredict.org/v1"

headers = {
    "Authorization": f"Bearer {api_key}",
    "Content-Type": "application/json"
}

# Predict cardiovascular risk
data = {
    "biomarkers": {
        "crp": 2.1,
        "pf4": 12.5,
        "troponin_i": 0.012,
        "bnp": 45
    },
    "environment": "space",
    "mission_duration": 14
}

response = requests.post(f"{base_url}/predict", 
                        headers=headers, 
                        json=data)

result = response.json()
print(f"Risk Score: {result['risk_score']}")
print(f"Risk Level: {result['risk_level']}")
JavaScript
const apiKey = 'YOUR_API_KEY';
const baseUrl = 'https://api.cardiopredict.org/v1';

async function predictRisk(biomarkers, environment = 'space') {
  const response = await fetch(`${baseUrl}/predict`, {
    method: 'POST',
    headers: {
      'Authorization': `Bearer ${apiKey}`,
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      biomarkers,
      environment,
      mission_duration: 14
    })
  });
  
  const result = await response.json();
  return result;
}

// Usage
const biomarkers = {
  crp: 2.1,
  pf4: 12.5,
  troponin_i: 0.012,
  bnp: 45
};

predictRisk(biomarkers)
  .then(result => {
    console.log('Risk Score:', result.risk_score);
    console.log('Risk Level:', result.risk_level);
  });

Prediction API

Core endpoint for cardiovascular risk prediction using biomarker data.

Predict Cardiovascular Risk
/v1/predict
POST

Analyzes biomarker profile and returns comprehensive cardiovascular risk assessment with recommendations.

Request Parameters
Parameter Type Required Description
biomarkers Object Required Key cardiovascular biomarker values
environment String Optional Environment type: "space", "bedrest", "hospital"
mission_duration Integer Optional Duration in days (for temporal modeling)
model_type String Optional "clinical" (default) or "research"
Biomarker Parameters
Biomarker Key Unit Normal Range
C-Reactive Protein crp mg/L < 3.0
Platelet Factor 4 pf4 ng/mL 4-20
Troponin I troponin_i ng/mL < 0.04
B-type Natriuretic Peptide bnp pg/mL < 100
Fetuin A36 fetuin_a36 Î¼g/mL 200-400
